SpaceTimeComponent#

class SpaceTimeComponent(params)#

A component that fits to the inferred origin of the samples.

The implementation of this class will closely follow that of Paper 1. However, the component object does not consider uncertainties.

Parameters

params (tuple) – A tuple of all parameters needed to define the component. Can be None as parameters can be set through calling set_parameters() or maximize() .

__init__()#

Constructor method

configure(cls, minimize_method='brent', reg_covar=1e-6, trace_orbit_func=trace_epicyclic_orbit, morph_cov_func=remove_posvel_correlations, **kwargs)#

Class method. Set class level configuration parameters. This must be called (even with just empty argument list) before objects can be instantiated. Defaults stored in method signature.

Parameters

  • minimize_method (str) – method used by scipy.optimize.minimize_scalar for optimizing the _loss(). Default ‘brent’.

  • reg_covar (float) – A regularization constant added to the diagonal elements of covariance matrices to ensure invertability. Default 1e-6.

  • trace_orbit_func (Callable of form (start: [n_dim], age: float, *args) -> end: [n_dim]) – A function that traces an orbit from a starting point in feature space through time by a certain age. Default trace_epicyclic_orbit.

  • morph_cov_func (Callable of form (approx_birth_covariance: [n_dim, n_dim]) -> birth_covariance: [n_dim, n_dim]) – A function that applies birth-site assumptions to the feature-space covariance matrix. Default remove_posvel_correlations.

estimate_log_prob(X)#

Estimate the log probability of each sample given the component. If the component is a simple Gaussian, this method would evaluate the Gaussian at the position of the sample. The features and how a component interprets them are unrestricted. Age, chemical composition etc. can all be in the feature, and this method determines how the component treats them.

Parameters

X – Input data

Returns

log probabilities of each sample
